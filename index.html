How It Works
Simple, Fast, Accessible
01
📷

Deaf User Signs

The deaf user performs a sign in front of the camera. MediaPipe tracks hand landmarks and translates the gesture into text or speech in real time.

02
🎙️

Hearing User Speaks

The hearing user speaks naturally or types a message. The system converts speech to text via Google STT, then maps it to sign language visuals.

03
⚡

Instant Translation

Both users receive translated content immediately — no sign language knowledge required from either side.
